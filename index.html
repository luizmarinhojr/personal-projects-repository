Últimos projetos

Manipulação de Lista Ordenada (ol): "Lista de Passos Interativa"

Crie uma lista ordenada com 5 passos para realizar uma tarefa (ex.: "Passos para fazer um bolo"). Ao clicar em cada item, ele deve mudar de cor e ser riscado, indicando que o passo foi concluído. • Use CSS para estilizar a lista de forma agradável

Manipulação de Lista Não Ordenada (ul): "Lista de Compras com Remoção de Itens"

Monte uma lista não ordenada com itens de compras. • Ao clicar em um item, ele deve ser removido da lista (simulando que o item foi comprado). • Estilize com CSS para deixar a lista visualmente destacada.

Manipulação de Tabela: "Tabela com Destaque de Linhas ao Clicar"

Crie uma tabela com pelo menos 4 linhas (ex.: lista de alunos com notas). • Ao clicar em uma linha, ela deve ser destacada com cor de fundo diferente (indicando seleção). • Use CSS para estilizar a tabela com bordas, espaçamento e transições.

Manipulação de Imagem: "Troca de Imagem ao Passar o Mouse"

Coloque uma imagem central na página. • Ao passar o mouse sobre a imagem, ela deve trocar para outra imagem. • Ao tirar o mouse, ela volta para a imagem original. • Estilize a imagem com bordas arredondadas e sombra.

Painel Interativo de Cartões (CSS + JavaScript)

Crie uma página com 3 cartões (div), cada um com um título e uma descrição. • Utilize CSS para estilizar os cartões (borda, sombra, padding, etc.). • Ao passar o mouse sobre o cartão, ele deve aumentar de tamanho e mudar de cor (efeito hover). • Use JavaScript para que, ao clicar em um cartão, a descrição seja trocada por "Você clicou neste cartão!".

Formulário de Validação de Email (CSS + JavaScript)

Monte um formulário simples com: • Um campo de entrada de email. • Um botão "Validar". • Utilize JavaScript para verificar se o email digitado possui o caractere "@" e um domínio (exemplo: nome@dominio.com). • Se o email for válido, exiba uma mensagem de sucesso em verde abaixo do campo. Caso contrário, exiba uma mensagem de erro em vermelho. • Use CSS para colorir as mensagens de acordo com o resultado.

To-Do List (Lista de Tarefas) Dinâmica (HTML + CSS + JavaScript)

Crie uma página com: • Um campo de entrada para adicionar uma nova tarefa. • Um botão "Adicionar". • Uma lista (ul ou ol) para exibir as tarefas. • Ao clicar em "Adicionar", a tarefa deve aparecer na lista. • Ao clicar em uma tarefa da lista, ela deve ser riscada (text-decoration: linethrough) para indicar que foi concluída. • Use CSS para deixar a lista organizada e com estilo agradável.

Jogo de Cores Aleatórias (CSS + JavaScript)

Monte uma página com: • Um botão "Gerar Cor Aleatória". • Uma área quadrada (div) com tamanho fixo (ex: 200px x 200px). • Ao clicar no botão, a área deve mudar para uma cor de fundo aleatória (usando JavaScript). • Mostre o código da cor gerada (ex: #A1B2C3) abaixo da área colorida. • Estilize a área com borda e sombra (CSS).

Modal Simples com CSS e JavaScript (Janela Flutuante)

Crie uma página com: • Um botão "Abrir Modal". • Uma janela modal (div) escondida inicialmente, com mensagem "Bem-vindo ao Modal!" e um botão "Fechar". • Ao clicar em "Abrir Modal", o modal deve aparecer no centro da tela com fundo semi-transparente. • Ao clicar em "Fechar", o modal deve sumir. • Use CSS para centralizar e estilizar o modal (sombra, arredondamento, fundo escuro transparente para a tela).

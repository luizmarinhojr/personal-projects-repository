6) Formulário de Validação de Email (CSS + JavaScript)

Monte um formulário simples com:

  • Um campo de entrada de email.
  • Um botão "Validar".
  • Utilize JavaScript para verificar se o email digitado possui o caractere "@" e um domínio (exemplo: nome@dominio.com).
  • Se o email for válido, exiba uma mensagem de sucesso em verde abaixo do campo. Caso contrário, exiba uma mensagem de erro em vermelho.
  • Use CSS para colorir as mensagens de acordo com o resultado.

7) To-Do List (Lista de Tarefas) Dinâmica (HTML + CSS + JavaScript)

Crie uma página com:

  • Um campo de entrada para adicionar uma nova tarefa.
  • Um botão "Adicionar".
  • Uma lista (ul ou ol) para exibir as tarefas.
  • Ao clicar em "Adicionar", a tarefa deve aparecer na lista.

  • Use CSS para deixar a lista organizada e com estilo agradável.
